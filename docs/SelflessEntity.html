Class: SelflessEntity

SelflessEntity(sdkParentEntity, entity)

Abstract Entity class to help create entity classes without self-links. These entities are transient, and are not saved within the Entity Store.

Constructor

new SelflessEntity(sdkParentEntity, entity)

Sets the parent object and setups the siren entity ready to be decoded by extending this class.
Parameters:
Name Type Description
sdkParentEntity Object Parent entity that is a Entity.
entity Object A hypermedia siren entity as defined by the siren specification
